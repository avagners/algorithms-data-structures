# Анализ работы цикла с двумя pop()

```java
while (stack.size() > 0) {
    stack.pop();
    stack.pop();
}
```

- Цикл извлечёт все элементы по одному
- Если в стеке нечетное кол-во, то последний `pop()` вернёт `null`
- Ошибки времени выполнения **не возникнет** (реализация возвращает `null` для пустого стека)
