# Рефлексия по заданию: Хэш-таблицы

## 3. Динамическая хэш-таблица

Моя реализация частично соответствует рекомендации. Предлагается использовать композицию с DynArray для управления памятью, тогда как моя реализация использует прямой массив с ручным управлением.

## 5. DDoS-атака на хэш-таблицу и защита солью

Моя реализация частично соответствует рекомендации.
Одна соль на всю таблицу не защищает от целенаправленной DDoS-атаки - атакующий может заранее подобрать коллизии для известной соли.
Нужно добавить динамическую соль на каждое значение.
